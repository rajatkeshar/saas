# saas-tutorial

To run this Node.js SaaS app starter code, you'll need to create a .env file in the project root directory and define these environment variables:  

AWS_KEY
AWS_SECRET
AWS_BUCKET_NAME
UPLOAD_FILE_SIZE_LIMIT
AWS_HOST_URL
SMTP_HOST
SMTP_PORT
SMTP_EMAIL
SMTP_PASSWORD
RAZORPAY_KEY
RAZORPAY_SECRET
